Hevean CS2 Blocker

A lightweight batch script that completely locks CS2 on your PC — via firewall rules, port blocking, and renaming the executable. Steam keeps working normally. Great for focus, parental control, or just going cold turkey.

Firewall rules (in & out) Port blocking (TCP & UDP) Renames cs2.exe Steam stays functional No install needed

How to use

  1. 1 Download Hevean.bat below
  2. 2 Close Steam and CS2 completely first
  3. 3 Right-click the file → "Run as administrator"
  4. 4 Done — CS2 is now fully blocked
Download Hevean.bat
Must be run as Administrator. If Windows SmartScreen appears, click "More info → Run anyway".
